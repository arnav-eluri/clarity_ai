# Clarity AI — Critical Thinking Companion

Clarity AI is a comprehensive digital wellness platform designed to help users navigate the complexities of online information and maintain mental well-being in the digital age.

## 🚨 Problem Statement
In an age of information overload, rapid news cycles, and algorithmic echo chambers, users frequently encounter misinformation, emotional manipulation, and cognitive fatigue. People struggle to quickly verify the credibility of content and often react impulsively to emotionally charged information. This leads to increased anxiety, polarized viewpoints, and poor digital habits.

## 💡 Solution We Used
We built **Clarity AI**, a digital wellness and media literacy platform that empowers users to slow down, analyze content, and reflect on their emotional responses. 
Our solution provides:
- **Real-Time Credibility Scanner**: Evaluates the credibility, logical fallacies, and biases of any pasted text.
- **Cognitive Load Analyzer**: Generates a score based on emotional intensity, complexity, and clarity.
- **Guided Reflection Mode**: An adaptive 5-step process (Pause, Summarize, Evaluate, Think, Decide) that helps users process their emotional reactions.
- **Supportive AI Chatbot**: A companion to guide users toward clearer, healthier digital thinking.

## 🏗️ Architecture
The architecture is designed to be lightweight, responsive, and easy to deploy:
- **Frontend**: Clean, minimalist UI built with HTML5, CSS3 (mobile-first), and Vanilla JavaScript.
- **Backend**: Python-based Flask server handling API routes and static file serving.
- **AI Integration**: Powered by OpenRouter API (using the `openai` SDK), routing prompts securely from the backend to advanced LLMs (like `gemini-2.0-flash-001`) for analysis, reflection generation, and chat responses.
- **Deployment**: Configured for standard WSGI hosting using Gunicorn and easily containerizable via Docker.
